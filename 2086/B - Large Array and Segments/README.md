<h3><a href="https://codeforces.com/contest/2086/problem/B" target="_blank" rel="noopener noreferrer">Large Array and Segments</a></h3>

<div class="header"><div class="title">B. Large Array and Segments</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>There is an array $$$a$$$ consisting of $$$n$$$ <span class="tex-font-style-bf">positive</span> integers, and a positive integer $$$k$$$. An array $$$b$$$ is created from array $$$a$$$ according to the following rules:</p><ul> <li> $$$b$$$ contains $$$n \cdot k$$$ numbers; </li><li> the first $$$n$$$ numbers of array $$$b$$$ are the same as the numbers of array $$$a$$$, that is, $$$b_{i} = a_{i}$$$ for $$$i \le n$$$; </li><li> for any $$$i > n$$$, it holds that $$$b_{i} = b_{i - n}$$$. </li></ul><p>For example, if $$$a = [2, 3, 1, 4]$$$ and $$$k = 3$$$, then $$$b = [2, 3, 1, 4, 2, 3, 1, 4, 2, 3, 1, 4]$$$. </p><p>Given a number $$$x$$$, it is required to count the number of such positions $$$l$$$ ($$$1 \le l \le n \cdot k$$$), for which there exists a position $$$r \ge l$$$, such that the sum of the elements of array $$$b$$$ on the segment $$$[l, r]$$$ is <span class="tex-font-style-bf">at least</span> $$$x$$$ (in other words, $$$b_{l} + b_{l+1} + \dots + b_{r} \ge x$$$).</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of several test cases. The first line contains one integer $$$t$$$ ($$$1 \le t \le 10^{4}$$$) — the number of test cases. The description of the test cases follows.</p><p>The first line of each test case contains three integers $$$n$$$, $$$k$$$, $$$x$$$ ($$$1 \le n, k \le 10^{5}$$$; $$$1 \le x \le 10^{18}$$$).</p><p>The second line of each test case contains $$$n$$$ integers $$$a_{i}$$$ ($$$1 \le a_{i} \le 10^{8}$$$).</p><p>Additional constraints on the input: </p><ul> <li> the sum of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^{5}$$$; </li><li> the sum of $$$k$$$ across all test cases does not exceed $$$2 \cdot 10^{5}$$$. </li></ul></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output one integer — the number of suitable positions $$$l$$$ in the array $$$b$$$.</p></div><div class="sample-tests"><div
